Character Sheet & Tutorial

When we first discussed the character sheet page, we wanted everything the player needed to know to be displayed in a orderly manner, with interactable buttons that allow the user to alter values, and take notes. However, a page on that scale would have taken too long to produce with a finished outcome, so instead we opted for a more plain layout that can also be displayed properly on mobile devices.

Initially, the tutorial was supposed to be a seperate page that gave the user a scripted walkthrough of the character sheet page, and required interaction with the page. However, this was deemed to be too difficult for us to implement, so instead, we decided to merge the tutorial page with the character sheet page. The product of this is a button to toggle the "tutorial mode", which enables pop-ups giving descriptions on key words that are present on the character sheet.
